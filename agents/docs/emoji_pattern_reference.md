## 0. 絵文字制御の基本

### 絵文字をテキストに挿入するだけで音声が変わる

```
通常テキスト: 「ねぇ、こっち向いて」
絵文字付き:   「👂 ねぇ、こっち向いて 💋」  ← 耳元で囁き＋リップノイズ
```

### サンドイッチ方式（推奨）
```
{prefix絵文字} テキスト {suffix絵文字}
```

prefix が「セリフ前の状態」、suffix が「セリフ後の余韻」を作る。

## 2. 強度別パターン（intensity guide）

### 重要な原則

> **絵文字は2-3個が最適バランス**。4個以上は不自然な音声・ノイズが増える。
> 例外：CFG=6.0 と組み合わせる場合のみ 4+4 が可。

### 強度マッピング

| 強度 | 構成 | CFG | guidance_mode | 用途 |
|------|------|-----|---------------|------|
| **弱** | 2+1 | 3.0 | joint | 日常・自信・威圧・強がり・挑発 |
| **中** | 2-3+1 | 5.0 | alternating | 恥じらい・困惑・喘ぎ(弱)・動揺・恐怖・抵抗 |
| **強** | 3+2 | 7.0 | alternating | 喘ぎ(強)・絶頂・叫び・嗚咽・荒い息・崩壊 |
| **極限** | 4+4 | 6.0 | alternating | 連続絶頂・出産・全穴同時・完全崩壊 |

### 強度ガイドの適用例

```
弱   :  🥵 「ふぅ……」 💦
中   :  🥵😮‍💨 （体が……熱い……） 💦
強   :  🥵😮‍💨💦 「あ゛ぁっ……イクっ……」 😭💦
極限 :  🥵😮‍💨💦😱 「ん゛ほぉぉっ……」 😭💦😱🫃
```

## 7. ❌ やりがちな失敗

### 1. 4個以上の絵文字を CFG=7 で使う
```
❌ 🥵🥵🌬️😱😮‍💨 「あ゛ぁっ！」 😭💦😱
```
→ 声がかすれて使い物にならない。**CFG=6 に下げる**か **絵文字を3個に減らす**。

### 2. 矛盾する絵文字の混在
```
❌ 🥵😪🐢⏩ 「ねぇ……」 💦
```
→ 喘ぎ・眠気・ゆっくり・早口、全部入って意味不明。意図を1つに絞る。

### 3. キャプションと絵文字が矛盾
```
caption: 「穏やかに話している」
text: 🥵😱💦 「ふぅ……」 😭💦
```
→ どっちが主か分からず崩れる。caption と絵文字の方向性を揃える。

### 4. 音響系を本文中に入れすぎ
```
❌ 「ね👂え👂、こ👂っち見て👂」
```
→ 文章途中の絵文字は不自然。**前後にサンドイッチ**するのが基本。

### 5. ⏸️ を使いすぎ
```
❌ 「あの⏸️でも⏸️わたし⏸️は⏸️」
```
→ 沈黙が多すぎてテンポ崩壊。1〜2箇所に絞る。

## 8. デバッグ Tips

### 思った効果が出ない場合

1. **CFG を上げる**（5.0→7.0）→ 絵文字の影響が強くなる
2. **絵文字をスタッキングする**（👂 → 👂👂）
3. **強度を上げる**（中→強で絵文字数を増やす）
4. **ノイズが出始めたら CFG 下げる**（7→6 + 4+4構成）

### キャラに合わせる時

1. クール系：絵文字数少なめ（弱〜中）、🥵 控えめ
2. 元気系：😆 や 😊 を多用、テンポ早め
3. 健気系：🥺 / 😭 / 🙏 / 🫣 を中心に
4. 淫乱系：🥵 / 😮‍💨 / 💦 / 🫃 を盛る
