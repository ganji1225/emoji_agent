# 台本生成依頼

## 作品概要
- タイトル: テスト作品
- ジャンル: 甘々・癒し系
- 目安尺: 5分
- 方向性: 全体的に甘い雰囲気。テスト用なので6行程度で短くまとめる。

## 登場人物

### ゆい
- 年齢: 20代前半
- 性格: 甘えん坊で素直、少し恥ずかしがり
- 口調: タメ口。語尾に「だよ」「ね」を多用。照れると声が小さくなる
- 体の特徴: 小柄で華奢
- 関係性: 恋人


## シーン構成

### scene_01: 日常パート
- 概要: 帰宅した恋人を出迎える。今日あったことを楽しそうに話す。
- 雰囲気・感情の流れ: リラックス→少しドキドキ
- 目安セリフ数: 3行

### scene_02: 甘えパート
- 概要: 恋人に甘えながら気持ちを伝える。
- 雰囲気・感情の流れ: 恥じらい→甘え→幸せ
- 目安セリフ数: 3行


## 出力形式

以下のCSV形式で出力してください。ヘッダー行を含めてください。

```csv
scene_id,speaker,emotion,text
```

各列の説明:
- scene_id: シーン区切り（scene_01, scene_02, ...）
- speaker: 話者名（ゆい）
- emotion: 感情・演技指示（日本語で簡潔に。例: 囁き, 照れ, 甘え, 喘ぎ, 喜び, 日常, 焦らし, 恥じらい 等）
- text: セリフテキスト（地の文は不要。1行1セリフ。）

## 注意事項
- 総セリフ数は約6行を目安にしてください
- 1行に1セリフのみ（地の文・ト書きは不要）
- emotion列には感情や演技の指示を日本語で記述してください
- 難読漢字は避け、ひらがな・カタカナを活用してください
- 句読点は適切に入れてください
- 「…」（三点リーダー）は躊躇や余韻の表現に使ってください
- セリフは自然な話し言葉で書いてください
