## 出力形式

**必ずJSON配列形式で出力してください。** CSV形式は使わないでください。

```json
[
  {
    "scene_id": "scene_01",
    "line_id": 1,
    "speaker": "キャラ名",
    "emotion_tags": "自信,威圧",
    "text_raw": "セリフテキスト",
    "emoji_suggestion": "😤💪 セリフテキスト 😤"
  }
]
```

### 各フィールドの説明

| フィールド | 説明 |
|---|---|
| `scene_id` | シーン区切り（scene_01, scene_02, ...） |
| `line_id` | シーン内の連番（1, 2, 3...） |
| `speaker` | 話者名 |
| `emotion_tags` | 感情タグ（カンマ区切り。下記ガイド参照） |
| `text_raw` | セリフテキスト（絵文字を含めない素のテキスト） |
| `emoji_suggestion` | 絵文字付きテキスト（サンドイッチ方式で前後に配置） |

### emoji_suggestion の書き方（サンドイッチ方式）

音声合成エンジンは**テキストの前後に絵文字を配置する「サンドイッチ方式」**で感情表現を制御します。
感情が強い場面では、前後に多くの絵文字を配置するほど感情表現が豊かになります。

**配置ルール:**
- 弱い感情: `"😤💪 テキスト 😤"` （前2-3個、後1個）
- 中程度: `"🥵😖🫣 テキスト 🫣"` （前2-4個、後0-1個）
- 強い感情: `"🥵😮‍💨💦 テキスト 😭💦"` （前3-5個、後2-3個）
- 極限状態: `"🥵😱😮‍💨💦🫃😭 テキスト 😱💦🥵"` （前5-8個、後2-3個）

**推奨絵文字パターン（場面別）:**

| 場面 | 前方（prefix） | 後方（suffix） |
|---|---|---|
| 恐怖・拒絶 | 😱 | 😱 |
| 恥じらい・困惑 | 🥵🫣 | 💦 |
| 低く太い喘ぎ（お゛ほぉ系） | 🥵😮‍💨💦 | 😭 |
| 鼻にかかった喘ぎ（ん゛ほぉ系） | 🫣🥵😰 | 💦 |
| 短く鋭い絶叫（あ゛あっ系） | 😱🥵💦 | 😱💦 |
| 甘え・懇願 | 🥵🙏 | 💦 |
| 懇願（もっと…して系） | 🙏🥵😭 | 💦 |
| 事後・放心 | 🥵😌💦 | 😌 |
| 絶望・崩壊 | 😭🥵💦 | 😱 |
| 極限崩壊 | 🥵😮‍💨💦😱 | 😭 |
| 耳元囁き | 👂🥵😮‍💨 | 💋 |

`emoji_suggestion` は空欄でもOKです（その場合、emotion_tagsから自動生成します）。

## emotion_tags の使い方ガイド

シーンに応じて以下の感情タグをemotion_tags列に記述してください。
音声合成エンジンがこのタグをもとに声の演技を制御します。

【基本感情】自信, 威圧, 挑発, 動揺, 恐怖, 絶望, 強がり, 驚き, 怒り
【恥じらい系】照れ, 恥じらい, 困惑, もじもじ
【甘え・親密系】甘え, 囁き, からかう, 優しい
【息遣い系】吐息, 喘ぎ, 喘ぎ(弱), 喘ぎ(強), 息切れ, 荒い息遣い
【泣き系】泣き, 嗚咽, 涙声
【極限系】絶頂, 叫び, 悲鳴, 苦しい, 崩壊
【事後系】放心, 脱力, 朦朧, 虚脱, 安堵
【拒否系】拒絶, 屈辱, 抵抗, 懇願

### 感情の強度と複数タグの使い方

感情表現が強い状態では、**emotion_tags に複数の感情タグをカンマ区切りで積極的に組み合わせてください**。
タグが多いほど音声合成時の感情表現が豊かになります。

- 弱い感情 → タグ1-2個: `"困惑"` `"動揺,恐怖"`
- 中程度 → タグ2-3個: `"喘ぎ(弱),困惑,恥じらい"` `"恐怖,強がり,涙声"`
- 強い感情 → タグ3-4個: `"喘ぎ(強),息切れ,荒い息遣い"` `"絶頂,叫び,喘ぎ(強),嗚咽"`
- 極限状態 → タグ4個以上: `"絶頂,叫び,喘ぎ(強),荒い息遣い,嗚咽"`

**特に激しい場面では、1つのemotion_tagsに3個以上のタグを使うことを推奨します。**

## 1行のテキスト量ガイド（TTS最適化）

音声合成では**1行あたり15-50文字**が最適です。
長いセリフは意味の区切りで分割してください。

| 種類 | 文字数 | 例 |
|---|---|---|
| 短い反応・叫び | 5-15文字 | `くっ……！` `放して……！` |
| 通常の発話 | 15-40文字 | 通常のセリフ |
| 長めの独白 | 40-60文字 | （括弧の内心独白） |
| **NG: 長すぎ** | 60文字超 | 必ず2行に分割 |

**分割のコツ:**
- 感情が変わるポイントで分割
- 「……」の後が自然な区切り
- 声に出す発話と（内心の独白）は別の行に
- 喘ぎ声は短く切る方が音声のリアリティが上がる

## 注意事項
- 総セリフ数は約{{目安セリフ数}}行を目安にしてください
- **必ずJSON配列形式で出力してください**（CSVではなく）
- text_raw にはセリフのみ記述（絵文字は含めない。絵文字はemoji_suggestionに書く）
- 1行に1セリフのみ（地の文・ト書きは不要）
- **1行あたり15-50文字を目安に**（長い場合は分割）
- 難読漢字は避け、ひらがな・カタカナを活用
- 句読点は適切に入れる
- 内心の独白は（括弧）で表現: （これは……瘴気？）
- 発話は鉤括弧なしで記述: やめて……離れなさい……っ！
