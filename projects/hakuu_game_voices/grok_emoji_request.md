# 絵文字（emoji）追加依頼 - ゲームテキスト音声化用

## 概要

以下に添付するゲームテキスト（バッドエンディング＋イベント失敗テキスト）から、
**独白（括弧の行）と発話（鉤括弧の行）のみ** を抽出し、
音声合成（TTS）用の絵文字付きテキストに変換してください。

## 対象ファイル
- `hakuu_bad_end_text.md` — バッドエンディング6シーン
- `hakuu_failure_text.md` — イベント失敗テキスト14イベント

## 抽出ルール

### 採用する行
- `（括弧で囲まれた独白）` → そのまま採用
- `「鉤括弧で囲まれた発話」` → 鉤括弧を外して採用

### 除外する行
- `【背景：...】` `【CG：...】` `【成人向け描写エリア】` → 演出指示、除外
- `★パラメータ +20` → ゲーム処理、除外
- `——END——` `---` → 区切り、除外
- `#` で始まるセクションヘッダ → scene_id に使用
- 地の文・ナレーション（括弧でも鉤括弧でもない行）→ 除外

## 出力形式

**JSON配列形式**で出力してください。

```json
[
  {
    "scene_id": "s1_mid",
    "line_id": 1,
    "speaker": "白雨",
    "emotion_tags": "絶望,恐怖",
    "text_raw": "（……くっ……体が……動かない……）",
    "emoji_suggestion": "😭😱 （……くっ……体が……動かない……） 😱"
  }
]
```

### scene_id の対応表

| セクションヘッダ | scene_id |
|---|---|
| バッドED S1-mid「孵化室の苗床」 | `s1_mid` |
| バッドED S1-boss「女王の器」 | `s1_boss` |
| バッドED S2-mid「飛行回廊の繁殖体」 | `s2_mid` |
| バッドED S2-boss「飛行回廊の花嫁」 | `s2_boss` |
| バッドED S3-mid「蔓の揺りかご」 | `s3_mid` |
| バッドED S3-boss「最深部の花」 | `s3_boss` |
| S1-ev1〜S3-boss-ev2 | そのまま小文字化（例: `s1_ev1`, `s3_boss_ev2`） |

## emotion_tags ガイド

以下の感情タグをカンマ区切りで指定してください。

【基本感情】自信, 威圧, 動揺, 恐怖, 絶望, 強がり, 驚き
【恥じらい系】恥じらい, 困惑
【甘え・親密系】甘え, 囁き
【息遣い系】喘ぎ(弱), 喘ぎ(強), 息切れ, 荒い息遣い
【泣き系】嗚咽, 涙声
【極限系】絶頂, 叫び, 崩壊
【事後系】放心, 虚脱, 安堵
【拒否系】恐怖, 拒絶

### 強度に応じたタグ数

- 弱い感情 → 1〜2個: `"困惑"` `"恐怖,拒絶"`
- 中程度 → 2〜3個: `"喘ぎ(弱),恥じらい,困惑"`
- 強い感情 → 3〜4個: `"喘ぎ(強),絶頂,荒い息遣い,嗚咽"`
- 極限 → 4個以上: `"絶頂,叫び,喘ぎ(強),荒い息遣い,嗚咽"`

## emoji_suggestion の書き方（サンドイッチ方式）

テキストの**前後に絵文字を配置**してください。感情の強さに応じて絵文字の量を変えます。

**配置ルール:**
- 弱い感情: `"😱 テキスト"` （前1-2個のみ）
- 中程度: `"🥵🫣 テキスト 🫣"` （前2-3個、後0-1個）
- 強い感情: `"🥵😮‍💨💦 テキスト 😭💦"` （前3-5個、後2-3個）
- 極限: `"🥵😱😮‍💨💦😭 テキスト 😱💦🥵"` （前5-8個、後2-3個）

**推奨パターン:**

| 場面 | prefix | suffix |
|---|---|---|
| 恐怖・拒絶 | 😱😭 | 😱 |
| 恥じらい・身体反応 | 🥵🫣 | 💦 |
| 低く太い喘ぎ（お゛ほぉ系） | 🥵😮‍💨💦 | 😭 |
| 鼻にかかった喘ぎ（ん゛ほぉ系） | 🫣🥵😰 | 💦 |
| 短く鋭い絶叫（あ゛あっ系） | 😱🥵💦 | 😱💦 |
| 懇願（もっと…して系） | 🙏🥵😭 | 💦 |
| 事後・放心 | 🥵😌💦 | 😌 |
| 絶望・崩壊 | 😭🥵💦 | 😱 |

## テキスト量の注意

- 1行あたり **15〜55文字** が最適
- 60文字を超える行は意味の区切り（「……」の後）で分割してください
- 喘ぎ声は短く切る方が音声のリアリティが上がります

## 重要な注意事項

- **ナレーション（地の文）は絶対に含めないでください** — 独白と発話のみ
- text_raw は原文をそのまま保持してください（絵文字は emoji_suggestion のみ）
- バッドED の【前半】【後半】の区別は不要です（同じ scene_id 内で line_id を連番）
- failure テキストの `★パラメータ` 行は無視してください
